Enumerations

e_i2c_direction

Request type, read or write

I2C_DIR_WRITEwrite request
I2C_DIR_READread request

e_i2c_frequency_mode

SCL frequency ranges defined in the I2C standard.

I2C_STANDARD_MODE0 : 100KHz
I2C_FAST_MODE0 : 400KHz
I2C_FAST_MODE_PLUS0 : 1MHz
I2C_HIGH_SPEED_MODE0 : 3.4MHz

Macros

I2C_DEFAULT_SLAVE_ADDRESS

Default address set for slave agents.  Master agents receive this value as the default valid slave address in the environment.  Value = ‘h52.
