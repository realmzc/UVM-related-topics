Constraints

agent_is_active_c

Constraints variable i2c_cfg::is_active.  Default: is_active = UVM_ACTIVE.

address_bits_c

Constraints variable i2c_cfg::address_num_of_bits.  Define the number of address bits used.  Valid values are 7 or 10.  Default: address_num_of_bits == 7.

requested_scl_frequency_in_khz_c

Constraints variable requested_scl_frequency_in_khz.  Sets SCL frequency for masters. dependent on i2c_cfg::frequency_mode_range value.

max_num_of_master_words_c

Constraints variable max_num_of_master_words.  Default allows a maximum of 10 transmission words per address.

valid_slave_address_c

Constraints queue valid_slave_address to the default slave address for the initial integration phase.  Basic setup has a single slave constrained to address i2c_package::I2C_DEFAULT_SLAVE_ADDRESS.  This constraint is used only for the initial integration phase and can be ignored when setting the environment I2C addresses.
