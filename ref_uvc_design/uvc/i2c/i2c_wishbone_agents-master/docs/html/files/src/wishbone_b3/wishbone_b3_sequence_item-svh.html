Variables

direction_e

rand e_wishbone_b3_direction direction_e

defines if the request is a read or write.  See wishbone_b3_package for valid enumerations.

address

rand logic [ADR_W-1:0] address

requested address.  A master agent will drive the address bus with this value while a slave agent will receive the value on the address bus.

data

rand logic [DAT_W-1:0] data

transaction data value.  A master agent sets this members value for a write request and receives a value for a read request.

select

rand logic [(DAT_W/8)-1:0] select

byte select.  Each select bit represents a valid data byte in the data member.

response_e

rand e_wishbone_b3_response response_e

response notifying if the transaction was successful.  See wishbone_b3_package for valid enumerations.
